🤖 Corpus AI Companion

Complete API Documentation for Raspberry Pi AI System

📋 System Overview

Corpus is a modular AI companion system that provides your Raspberry Pi with multiple sensory and cognitive capabilities. Each capability runs as an independent microservice with its own API.

🎛️ Orchestrator API
Main coordination system that manages all capabilities and provides unified command interface.
Port: 5000
📖 Swagger Docs 📊 Status
curl http://raspberrypi:5000/status
🗣️ Speech API
Text-to-speech synthesis using pyttsx3. Converts text into natural speech output.
Port: 5001
📖 Swagger Docs 📊 Status
curl -X POST http://raspberrypi:5001/speak -H "Content-Type: application/json" -d '{"text":"Hello!"}'
👁️ Vision API
Computer vision using Logitech 4K camera + Gemini AI. Captures and analyzes images with first-person descriptions.
Port: 5002
📖 Swagger Docs 📊 Status
curl http://raspberrypi:5002/describe

🚀 Quick Start Examples

Make Your Pi Speak

curl -X POST http://raspberrypi:5001/speak -H "Content-Type: application/json" -d '{"text":"I am alive and ready to help!"}'

Get Current View Description

curl http://raspberrypi:5002/describe

Start Continuous Vision (every 10 seconds)

curl -X POST http://raspberrypi:5002/start_loop -H "Content-Type: application/json" -d '{"interval":10}'

Check All System Status

curl http://raspberrypi:5000/status
